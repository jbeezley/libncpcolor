Detailed Description

template<typename T>
class LinearNorm< T >

Defines an affine mapping from [minValue(),maxVal()] -> [0,255].

Template Parameters
Tspecifies the type that the norm acts on... the norm itself always returns a uint8_t.

basic.cpp.
