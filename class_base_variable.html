Detailed Description

A simplified wrapper around a netCDF variable object.

This is an abstract base class defining a netCDF variable. It's main purpose is to store the following:

  • The variable's name
  • The variable's shape and total size
  • The names of the variable's dimensions

The class contains the method readSlice which takes in a SliceType object and returns the corresponding data from the netCDF file.
